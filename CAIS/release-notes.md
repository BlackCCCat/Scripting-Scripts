## 本次更新

- 新增更新内容提示：CAIS 会在更新说明内容变化后自动弹出一次，关闭后不再重复提示。
- 自定义长按功能新增“网络请求”类型，支持通过 `request(text)` 发起 `fetch` 请求，并可控制返回结果是否写入系统剪贴板。
- 文本编辑页接入 Editor 原生搜索入口，可在编辑时打开搜索与替换面板。
- 设置页新增 iCloud 同步入口，可选择同步数据库和图片文件。

## 使用提示

- 网络请求脚本可直接使用当前条目内容参数 `text`，也可以在字符串中使用 `{{text}}`。
- 更新说明只根据 Markdown 内容变化判断是否再次显示；脚本版本号变化但说明内容不变时不会重复弹出。
